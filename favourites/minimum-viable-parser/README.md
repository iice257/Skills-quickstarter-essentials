# Minimum-Viable-Parser

A general-purpose routing skill that decides the lightest acquisition and parsing path before an AI agent reads, scrapes, parses, transforms, downloads, OCRs, screenshots, or browser-automates a source.

The skill prioritizes speed and low setup cost while preserving the accuracy required by the task. It prefers existing context, metadata probes, structured sources, static reads, and format-native parsers before escalating to OCR, Playwright, browser automation, or new dependency installs.
