# Debugging

Skills for browser checks, screenshots, CI failures, and PR feedback.

Paste into your agent:

```text
$skill-installer install the skills listed at https://github.com/iice257/Skills/tree/main/categories/debugging
```

## Skills

| Skill | Why |
| --- | --- |
| `playwright` | Browser automation and verification. |
| `playwright-interactive` | Iterative browser debugging. |
| `screenshot` | Desktop and app screenshot capture. |
| `gh-fix-ci` | Failed GitHub Actions checks. |
| `gh-address-comments` | Review comments and issue feedback. |
| `security-best-practices` | Security-aware bug review. |
