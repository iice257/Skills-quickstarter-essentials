# Categories

Scenario bundles for installing a focused set of skills.

Each category has:

- `README.md`: human-readable purpose and install prompt.
- `paths.txt`: canonical install paths under `all/`.

Paste a category link into your agent with `$skill-installer`, for example:

```text
$skill-installer install the skills listed at https://github.com/iice257/Skills/tree/main/categories/lightweight-quickstart
```

## Available Bundles

| Bundle | Purpose |
| --- | --- |
| `lightweight-quickstart` | Small starter pack. |
| `full-startup` | Everything in `all/`. |
| `web-development-setup` | Frontend, design, testing, CI, and deploy. |
| `debugging` | Browser, screenshots, PR checks, and review workflows. |
| `cybersec` | Security review and ownership mapping. |
